# 94. 二叉树的中序遍历

> https://leetcode-cn.com/problems/binary-tree-inorder-traversal/

# 递归解法


```js
var inorderTraversal = function(root) {
    let res = []
    if(!root) return res
    function inorder(root) {
        root.left && inorder(root.left)
        res.push(root.val)
        root.right && inorder(root.right)
    }
    inorder(root)
    return res
};
```
