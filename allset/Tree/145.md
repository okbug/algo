# 145. 二叉树的后序遍历

> https://leetcode-cn.com/problems/binary-tree-postorder-traversal/

# 递归解法

```js
var postorderTraversal = function(root) {
    let res = []
    if(!root) return res
    postOrder(root)
    function postOrder(root) {
        root.left && postOrder(root.left)
        root.right && postOrder(root.right)
        res.push(root.val)
    }
    return res
};
```
